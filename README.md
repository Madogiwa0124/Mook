# Webページ更新管理ツール"Moook"
このリポジトリは、Webページ更新管理ツール"Moook"のリポジトリです。  
Moookを使えば、webページの更新情報を一括管理出来ます。  

## Moookで出来ること
主な機能は下記です！
1. 更新確認を行いたいwebページを登録
2. 1時間おきに登録したページの更新有無を確認
3. 更新されていれば通知
